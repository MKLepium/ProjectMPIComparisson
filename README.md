## How to run

To run the program, run:

```bash
make MPI_IMPL=MPICH run_mpich
```
Or: 
```bash
make MPI_IMPL=OPENMPI run_openmpi
```

Or just:
```bash
make run_both
```


## Prerequisites

- C++ compiler
- OpenMPI and MPICH libraries
- mpiexec
- make
